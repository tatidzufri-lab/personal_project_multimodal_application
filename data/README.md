# Демонстрационные материалы

Эта папка содержит скриншоты и видеозаписи, демонстрирующие работу приложения "Мониторинг конкурентов - AI Ассистент".

## Содержимое

### Видеозаписи работы приложения

- **01_demo_text_analysis.mov** - Демонстрация анализа текста конкурента
- **02_demo_image_analysis.mov** - Демонстрация анализа изображения
- **03_demo_site_parsing.mov** - Демонстрация парсинга сайта
- **04_demo_history.mov** - Демонстрация работы с историей запросов

### Скриншоты интерфейса

- **01_main_screen.png** - Главный экран приложения
- **02_text_analysis_results.png** - Результаты анализа текста
- **03_image_analysis_results.png** - Результаты анализа изображения
- **04_site_parsing_interface.png** - Интерфейс парсинга сайта

## Как использовать

Все материалы можно просмотреть напрямую на GitHub или скачать для локального просмотра. Видеофайлы демонстрируют полный процесс работы с приложением, а скриншоты показывают ключевые моменты интерфейса и результатов анализа.
